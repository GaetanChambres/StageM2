csv file with name DSxfeatures contains :

filename , start time , end time , mfcc coef x13 , mfcc bands x 40 , crackle , wheezle

details :

filename -- name of the wav record

start time -- begining of the respiratory cycle

end time -- end of the respiratory cycle

mfcc coef -- the 13 mfcc coefs computed on the respiratory cycle

mfcc bands -- the 40 mffc bands computed on the respiratory cycle

crackle -- 1 if there are crackles , 0 if not

wheezle -- 1 if there are wheezles , 0 if not
